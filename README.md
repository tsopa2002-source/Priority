# PriorityCash v2

Изменения:
- Исправлен сброс клавиатуры: приложение больше не перерисовывает экран при каждом символе.
- Более дружелюбный интерфейс.
- Добавлены пояснения приоритетов.
- Названия и описания приоритетов редактируются.
- Можно добавлять новые строки в бюджет.
- Данные каждого пользователя хранятся локально на устройстве.


## Обновление без ручной чистки кэша

Версия 2.1.0 добавляет автоматическое обновление service worker:
- новая версия принудительно активируется через `skipWaiting()`;
- старые кэши `prioritycash-*` удаляются при активации;
- навигация и основные файлы берутся по network-first стратегии;
- страница перезагружается при смене service worker.

Обычно после загрузки новой версии на Netlify достаточно открыть приложение снова. Чистить кэш вручную не нужно.
