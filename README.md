# PriorityCash v3

- Добавлены настройки пользователя: ник, тема, названия приоритетов.
- Тема: светлая, тёмная или как в телефоне.
- Поле “комната” заменено на настраиваемую регулярную цель.
- После изменения дохода/расходов/цели остатки в карточках месяца пересчитываются сразу.
- Добавлен календарь остатка по дням: плановый и фактический прогноз.
- Возможность добавлять строки бюджета сохранена.
- Service worker обновляется автоматически без ручной чистки кэша.
