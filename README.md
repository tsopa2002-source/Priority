# План машины и комнаты: PWA для iPhone

Это статическое веб-приложение без сервера. Данные хранятся в браузере iPhone.

## Как открыть на iPhone

1. Залей папку на любой HTTPS-хостинг: GitHub Pages, Netlify, Vercel, свой сервер.
2. Открой `index.html` в Safari.
3. Нажми **Поделиться** → **На экран Домой**.
4. Запускай как обычное приложение.

## Что умеет

- Ввод дохода, расходов и суммы на комнату по месяцам.
- Автоматический водопад денег: A → B → C → D.
- Лимит комнаты.
- Редактирование бюджета работ.
- Экспорт/импорт JSON.
- Экспорт CSV для Excel.
- Работает офлайн после первой загрузки через service worker.

## Важно

Данные хранятся локально в браузере. Перед очисткой Safari или переустановкой телефона сделай экспорт JSON.
